Find closest locations by drive time

ℹ️ About this tool

Created by: Olivier

This tool helps you find the closest dealer locations by drive time.

  • The backend now runs on Flask + PostgreSQL and supports Slack slash commands for managing dealers.
  • New Slack flows: /add_dealer and /dealer_edit open modals for dealer creation and updates this is available to a specific channel only.
  • Edited dealers refresh coordinates when the address changes, and all changes are saved to the database.
  • Slack now posts public channel updates showing who added or edited a dealer and what changed.
  • Geocoding uses Nominatim first, then OpenRouteService as a fallback, with caching in the database and memory.
  • Auto-complete is still US-only, and drive-time results are approximate.
  • The app is more stable and now centralizes dealer data in the database instead of static Excel only, searches are saved for future use as well.
